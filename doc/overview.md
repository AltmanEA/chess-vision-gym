# Обзор проекта шахматного тренажера для ИИ-помощника

## 1. Краткое описание проекта

Подробное описание проекта содержится в следующих документах:
- spec.md - полная спецификация проекта
- plan.md - план реализации

## 2. Текущее состояние кодовой базы

### 2.1. Основной компонент (src/App.tsx)
- Реализует отображение шахматной доски через react-chessboard
- Поддерживает загрузку позиций из FEN строк
- Обрабатывает перемещение фигур через drag-and-drop
- Содержит поле ввода для загрузки FEN позиции

### 2.2. Утилиты для работы с FEN (src/utils/fenUtils.ts)
- validateFen: проверка корректности FEN строки
- loadPositionFromFen: создание объекта Chess из FEN
- setInitialPosition: установка начальной позиции

### 2.3. Зависимости
- chess.js: шахматная логика
- react-chessboard: отображение доски
- @dnd-kit/core, @dnd-kit/modifiers: drag-and-drop функциональность
